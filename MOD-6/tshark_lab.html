[ TSHARK -TRAFFIC ANALYSIS ]

DESAFIO CTF: PCAP FORENSICS

Analise o arquivo http.cap utilizando o TShark para responder às perguntas.

#01 - Identificação do Servidor

Qual é o endereço IP de destino do servidor para o qual o navegador web realiza a primeira requisição GET HTTP?

Use TShark:
tshark -r http_with_jpegs.cap -Y "http.request.method == GET" -T fields -e ip.dst | head -n 1

#02 - Impressão Digital do Client

O atacante/usuário utilizou um navegador alternativo (Opera). Qual versão do produto Opera está explicitamente declarada no cabeçalho do User-Agent na primeira requisição?

Use TShark:
tshark -r http_with_jpegs.cap -Y "http.user_agent contains Opera" -T fields -e http.user_agent | head -n 1

#03 - Reconhecimento de Aplicação (Banner Grabbing)

Qual a string exata do banner de Server HTTP (Nome e Versão do Servidor + OS) retornada pelo servidor Apache na primeira resposta válida (HTTP 200 OK)?

Use TShark:
tshark -r http_with_jpegs.cap -Y "http.response.code == 200" -T fields -e http.server | head -n 1

#04 - Descoberta de Endpoint / Exfiltração

Qual o URI do script ASP acessado via método POST em direção ao host 'ins1.opera.com'?

Use TShark:
tshark -r http_with_jpegs.cap -Y "http.request.method == POST" -T fields -e http.request.uri

#05 - Content Carving (Análise de Mídia)

O arquivo HTML principal carrega uma imagem JPG de uma famosa cidade australiana. Qual é o nome do arquivo dessa imagem (incluindo a extensão)?

Use TShark:
tshark -r http_with_jpegs.cap -Y "http.request.uri contains jpg" -T fields -e http.request.uri
