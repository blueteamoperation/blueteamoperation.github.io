#01 - Mapeamento de Nós Ativos Menu: Endpoints

Acessando a lista global de Endpoints do Wireshark, quantos endereços IPv4 únicos geraram ou receberam tráfego nesta captura?

Caminho no Wireshark:
Vá no menu superior: Statistics -> Endpoints.
Selecione a aba IPv4 na janela e veja a quantidade de IPs exibidas na tabela.

#02 - Volumetria de Conversas TCP Menu: Conversations

Analisando as conexões de transporte no menu Conversations, qual foi a porta TCP de origem (porta alta do cliente) utilizada que gerou o maior números de bytes?

Caminho no Wireshark:
Acesse: Statistics -> Conversations.
Clique na aba TCP. Ordene os bytes do maior para o menor e observe a porta utilizada pelo cliente.

#03 - Título da Página Principal Filtro de Exibição

Ao analisar a primeira resposta HTTP bem-sucedida da raiz do servidor principal (10.1.1.1), qual é a string exata contida dentro da tag de título (<title>) do código HTML retornado?

Filtro sugerido: Insira na barra de filtros: http.response.code == 200
Selecione o primeiro pacote de resposta do IP 10.1.1.1. No painel inferior de inspeção de dados textuais/HTML (ou dando um Follow HTTP Stream), localize o texto que está entre <title> e </title>.
