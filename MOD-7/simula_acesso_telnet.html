Explore as vulnerabilidades de protocolos legados sem criptografia para auditar um servidor Linux remoto.

🎯 Cenário para o Aluno: Você capturou strings de texto em um tráfego de rede e conseguiu acesso ao prompt do Telnet de core-srv.hack.ru. Execute os comandos padrão de shell para coletar informações.
  • Passo 1: Autentique com o usuário capturado: root
  • Passo 2: Digite a senha administrativa vazada: telnet_is_dead_2026
  • Passo 3: Descubra em qual diretório você está usando o comando: pwd
  • Passo 4: Liste os arquivos ocultos do servidor com: ls -la
  • Passo 5: Leia o arquivo secreto para expor a flag: cat flag.txt
  • Passo 6: Finalize a sessão remota digitando: exit
