[ WINDOWS LIVE FORENSICS // COMMAND CONSOLE ]

LAB CTF: ARSENAL DE AQUISIÇÃO E TRIAGEM A QUENTE

1. Informações do Sistema

Nome do computador:
hostname
Usuário atual logado:
whoami
Informações detalhadas do S.O.:
systeminfo
Versão, build e especificação do Windows:
wmic os get Caption,Version,BuildNumber
Data e hora do sistema:
date /t & time /t

2. Usuários e Sessões

Verificar usuários ativos na sessão:
query user
Lista de usuários locais cadastrados:
net user
Membros do grupo Administradores locais:
net localgroup administrators
Privilégios e grupos completos do token atual:
whoami /all

3. Processos em Execução

Listagem básica de processos ativos:
tasklist
Processos associados a serviços do sistema:
tasklist /svc
Detalhamento completo dos processos (WMIC):
wmic process list full
Mapeamento de PID, PPID, Caminho e Linha de comando executada:
wmic process get ProcessId,ParentProcessId,ExecutablePath,CommandLine

4. Serviços

Status dos serviços ativos na máquina:
sc query
Lista simplificada de serviços ativos:
net start
Mapeamento estruturado de serviços, estado e binário correspondente:
wmic service get Name,State,StartMode,PathName

5. Conexões de Rede

Mapeamento de portas abertas, sockets ativos e PIDs:
netstat -ano
Conexões de rede com binário/executável associado (Requer Admin):
netstat -abno
Inspeção da tabela de cache ARP:
arp -a
Configuração completa de adaptadores de rede:
ipconfig /all
Tabela de roteamento de pacotes do sistema:
route print
Visualização do cache de nomes NetBIOS:
nbtstat -c
Compartilhamentos de rede externos ativos mapeados localmente:
net use

6. Compartilhamentos

Compartilhamentos locais ativos (SMB):
net share
Nome e caminhos físicos dos compartilhamentos locais ativos:
wmic share get Name,Path

7. Tarefas Agendadas

Listagem detalhada e verbose de tarefas agendadas no agendador do Windows:
schtasks /query /fo LIST /v

8. Drivers Carregados

Listar todos os drivers instalados no kernel:
driverquery
Exibição em modo detalhado (verbose):
driverquery /v
Exibir informações de assinatura digital de drivers:
driverquery /si

9. Evidências de Persistência

Chaves Run do Registro de Sistema (Local Machine):
reg query "HKLM\Software\Microsoft\Windows\CurrentVersion\Run"
Chaves Run do Registro de Sistema (Current User):
reg query "HKCU\Software\Microsoft\Windows\CurrentVersion\Run"

10. Firewall

Exibir status do firewall em todos os perfis de rede ativos:
netsh advfirewall show allprofiles
Listar todas as regras de entrada e saída ativas:
netsh advfirewall firewall show rule name=all

11. Logs de Eventos

Listar todos os arquivos/canais de logs de eventos do Windows:
wevtutil el
Consultar os últimos 20 eventos do log de Segurança em texto puro:
wevtutil qe Security /c:20 /f:text
Consultar os últimos 20 eventos do log do Sistema em texto puro:
wevtutil qe System /c:20 /f:text

12. Arquivos Recentes

Listar a pasta de Downloads ordenando do arquivo mais recente para o mais antigo:
dir C:\Users\%USERNAME%\Downloads /O:-D
Listar Área de Trabalho ordenando por data de modificação:
dir C:\Users\%USERNAME%\Desktop /O:-D
Listar de forma recursiva a pasta temporária do Windows ordenando por data:
dir C:\Windows\Temp /S /O:-D

13. DNS Cache

Exibir o cache local de resolução DNS (domínios visitados pelo host):
ipconfig /displaydns

14. Políticas e Configurações

Exibir objetos de diretivas de grupo (GPO) aplicados a este computador:
gpresult /r
Consultar as políticas de auditoria ativas de todas as subcategorias:
auditpol /get /category:*
