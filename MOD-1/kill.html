Simulador: Cyber Kill Chain (7 Estágios)

CENÁRIO DO ATAQUE:
Um grupo APT iniciou um ataque buscando dados financeiros. 1. Pesquisaram IPs da empresa no Shodan. 2. Criaram um script em Python que empacota um Ransomware. 3. O script foi enviado via link em uma mensagem de Teams. 4. Ao clicar, o navegador executou o código devido a um plugin desatualizado. 5. O código baixou e instalou um binário persistente na pasta System32. 6. O malware abriu um túnel DNS com o servidor do invasor. 7. Por fim, todos os servidores de backup foram criptografados.
